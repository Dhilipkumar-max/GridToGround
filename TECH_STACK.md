# Tech Stack & Dependencies

GridToGround is built with a curated selection of modern technologies to ensure performance, scalability, and a premium user experience.

## Core Framework
- **[React Router v7](https://reactrouter.com/)**: The foundation of the web app, providing routing and server-rendering capabilities.
- **[React 19](https://reactjs.org/)**: For building a highly interactive UI with modern hooks and patterns.

## AI & Intelligence
- **[Google Gemini 2.5 Flash](https://deepmind.google/technologies/gemini/)**: Used for image analysis and layout extraction.
- **[Puter.js AI](https://docs.puter.com/ai/)**: The orchestration layer that connects the app to state-of-the-art AI models.

## 3D & Graphics
- **[Three.js](https://threejs.org/)**: The industry standard for 3D graphics on the web.
- **[React Compare Slider](https://github.com/nerducate/react-compare-slider)**: A seamless way to showcase before-and-after transformations.

## Platform & Backend services
- **[Puter.js](https://puter.com/)**:
  - **Auth**: Simplified cloud-native user accounts.
  - **KV Storage**: High-performance key-value storage for project metadata.
  - **Hosting**: Direct deployment to the Puter cloud.

## Styling & UI Components
- **[Tailwind CSS v4](https://tailwindcss.com/)**: For rapid, consistent, and maintainable styling.
- **[Lucide React](https://lucide.dev/)**: For a beautiful, consistent set of icons.
- **Custom UI Components**: Neobrutalist-inspired buttons and panels.

## Development Tools
- **Vite**: Ultra-fast build tool and development server.
- **TypeScript**: For type-safe code and improved developer experience.
